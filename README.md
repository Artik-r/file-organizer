# File Organizer

A Python automation project that automatically organizes files into folders based on their file extensions.

## Features

- Automatically detects file types
- Creates folders if they don't exist
- Organizes Images, Documents, Videos, Music, Python files, Archives, and Others
- Uses Python's pathlib and shutil modules

## Technologies Used

- Python 3
- pathlib
- shutil
- os
